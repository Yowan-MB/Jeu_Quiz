Nom du projet : Quiz Master Description : Un site web interactif permettant aux utilisateurs de tester leurs connaissances via un système de quiz dynamique.

🛠 Installation
Pré-requis :

Python (3.x)

Flask (pip install flask)

Bootstrap (chargé via CDN)

Git (pour versionner le projet)

Instructions : 

1️⃣ Clone le projet via Git :
bash
git clone https://github.com/ton_nom_utilisateur/ton_projet.git
cd ton_projet

2️⃣ Installe les dépendances Python :
bash
pip install -r requirements.txt

3️⃣ Lance l’application Flask :
bash
python app.py

4️⃣ Ouvre ton navigateur et visite :
http://127.0.0.1:5000

🎮 Utilisation
Choisis un niveau de difficulté (Facile, Moyen, Expert)

Attendre 3minutes avant l'affichage des quesions

Réponds aux questions du quiz

Visualise ton score et tes erreurs

Recommence pour améliorer ton résultat ! 🔥

💡 Fonctionnalités
✅ Interface moderne grâce à Bootstrap 
✅ Intégration de Flask pour la gestion dynamique des questions 
✅ Animations fluides et effets visuels améliorant l’expérience utilisateur
